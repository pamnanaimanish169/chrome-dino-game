Chrome Dino Timeline

From hidden Easter egg to global phenomenon

Present
The Era Continues

Dino Game is omni-present

The game remains a popular browser-based game, accessible when a user loses internet connection.

Initially, it had limited support on mobile, but it was eventually improved and became a staple on both desktop and Android Chrome versions.

September 2015
The Pterodactyls Saga

Pterodactyls is introduced.

Pterodactyls are added as a feature in a browser update

The game was added as a fun distraction while users were offline.

December 2014
Older Devices Update

Support for Older Devices

The game is re-released with updated code to support older devices.

The T-Rex character is internally named "Project Bolan", a reference to Marc Bolan, lead singer of the band T. Rex.

September 2014
The Beginning

Dino Game is Born

The Chrome Dino game is released, designed by Sebastien Gabriel and his team.

Originally built as a fun Easter egg, the game quickly gained global popularity.
